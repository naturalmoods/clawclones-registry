# 🦞 ClawClones Public Registry

Welcome to the official community gateway for [ClawClones.com](https://clawclones.com). 

This repository serves as a **transparent, community-driven database** for nominating and tracking new forks, clones, and alternatives in the OpenClaw AI ecosystem.

## 🚀 How it Works

Instead of a closed database, we use this repository to keep the nomination process open to everyone.

1.  **Nominate**: A user submits a GitHub URL via the "Add a Claw" form on [ClawClones.com](https://clawclones.com).
2.  **Public Issue**: The nomination appears here as a GitHub Issue.
3.  **Community Review**: The community can discuss, upvote (using reactions), and test the suggested clone.
4.  **Integration**: Once vetted, our automated pipeline (powered by OpenRouter AI) analyzes the repo and adds it to the live dashboard.

## 🗳️ How to Nominate

The preferred way to nominate a project is through our website's interface to ensure all data is formatted correctly. However, you can also manually:

*   [Open a New Nomination Issue](https://github.com/naturalmoods/clawclones-registry/issues/new?template=nomination.md&labels=nomination&title=Nomination:+author/repo)
*   Describe why the project is a significant alternative to OpenClaw.
*   Mention its tech stack (Rust, Go, Zig, etc.) and primary focus (Efficiency, Security, Multi-agent).

## 🛡️ Eligibility Criteria

To be included in the main dashboard at ClawClones.com, a project should:
- Be related to the **OpenClaw** architecture or serve as an AI-powered dev-tool clone.
- Be **Open Source** (GitHub/GitLab).
- Strictly follow the **"No Crypto"** policy of the ecosystem.
- Focus on performance, local-first logic, or extreme hardware optimization.

## 📊 Status Labels

- `nomination`: A new project suggested by the community.
- `analyzing`: Our AI pipeline is currently assessing the repository.
- `integrated`: The project is now live on ClawClones.com.
- `rejected`: The project did not meet the ecosystem's technical or safety criteria.
